主文件界面 ​

相关源文件

  • src/assets/file.list.scss
  • src/views/main/Main.vue

主文件界面是 优享云盘 云存储应用程序的主要文件管理中心。该组件为用户提供了全面的文件操作功能，包括浏览、上传、组织和管理其存储空间内的文件和文件夹。它实现了一个基于表格的文件浏览器，集成了上传功能、搜索功能和上下文文件操作。

界面结构 ​

主文件界面在 Main.vue 中实现，由几个不同的部分组成，这些部分协同工作，提供完整的文件管理体验。

该界面在两种主要状态之间进行切换：有文件的文件列表视图和无文件时鼓励用户进行首次文件上传的空状态。

顶部操作栏 ​

操作栏提供了主要的操作按钮和搜索功能，位于界面的顶部。

操作栏包括：

  • 上传组件：带有自定义 http-request 处理程序的 el-upload
  • 批量操作：对 selectFileIdList 进行操作的按钮
  • 搜索面板：绑定到 fileNameFuzzy 的输入框，带有回车键处理程序
  • 刷新控件：触发 loadDataList() 的动画图标

文件显示系统 ​

文件列表使用 Table 组件来显示文件信息，具有自定义列模板和行选择功能。

表格配置 ​

表格使用 columns 数组中定义的三列布局：

  • 文件名：带有文件图标和操作的自定义作用域插槽
  • 最后更新时间：标准属性显示
  • 文件大小：带有大小格式化的自定义作用域插槽

tableOptions 配置启用了复选框选择并设置了高度调整。

文件操作管理 ​

单个文件操作 ​

每一行文件通过 showOp 机制在鼠标悬停时提供上下文操作。

批量操作 ​

批量操作与表格行选择填充的 selectFileIdList 数组配合使用：

  • 批量删除：使用逗号分隔的文件 ID 调用 delFileBatch()
  • 批量移动：使用 moveFolderBatch() 启动文件夹选择

文件和文件夹创建 ​

新建文件夹 ​

新建文件夹工作流使用内联编辑系统：

newFolder() 函数创建一个临时行，其中 showEdit: true 且没有 fileId。saveNameEdit() 函数根据 fileId 的存在情况决定是创建还是重命名。

数据加载和状态管理 ​

数据加载管道 ​

loadDataList() 函数作为核心数据获取机制：

该函数根据当前状态构建参数，并在非 “全部” 类别中有条件地移除 filePid。

状态变量 ​

关键的响应式状态变量管理界面：

变量类型用途
tableData对象包含分页文件列表和元数据
selectFileIdList数组跟踪用于批量操作的所选文件 ID
currentFolder对象包含 fileId 的当前目录上下文
category字符串活动的文件类别过滤器
editing布尔值跟踪是否有任何行处于编辑模式
fileNameFuzzy字符串用于文件过滤的搜索词
