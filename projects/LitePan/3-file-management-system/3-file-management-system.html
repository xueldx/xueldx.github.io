文件管理系统 ​

相关源文件

  • src/assets/file.list.scss
  • src/components/Navigation.vue
  • src/views/main/Main.vue

文件管理系统为 优享云盘 云存储应用提供了核心的文件浏览、组织和操作功能。该系统使用户能够以表格形式查看文件、浏览文件夹层级结构、执行文件操作，并与上传、预览和共享子系统集成。

系统架构 ​

文件管理系统围绕 Main.vue 组件构建，该组件作为文件操作的中央协调组件。它与导航、预览和共享组件集成，以提供全面的文件管理体验。

组件交互流程 ​

文件管理系统遵循结构化的数据流，其中主控制器在用户界面元素和后端服务之间进行协调。

文件操作系统 ​

文件管理系统通过一组协调的函数提供全面的增删改查操作，这些函数处理用户交互和 API 通信。

操作函数API 端点描述
加载文件loadDataList()/file/loadDataList分页并过滤获取文件列表
创建文件夹newFolder()/file/newFoloder在当前文件夹中创建新目录
重命名editFileName() → saveNameEdit()/file/rename重命名文件和文件夹
删除单个文件delFile()/file/delFile确认后删除单个文件
批量删除delFileBatch()/file/delFile删除多个所选文件
移动单个文件moveFolder()/file/changeFileFolder将文件移动到不同目录
批量移动moveFolderBatch()/file/changeFileFolder移动多个文件
下载download()/file/createDownloadUrl + /api/file/download创建下载 URL 并启动下载

文件表格界面 ​

文件表格通过结构化的展示系统提供了查看和与文件交互的主要界面。

导航集成 ​

导航系统通过与主文件界面的紧密集成，提供文件夹层级结构管理和面包屑导航。

搜索和过滤 ​

系统提供搜索功能，可在当前目录上下文中过滤文件。

空状态处理 ​

当没有文件时，系统会显示带有快速操作选项的空状态界面。

API 通信模式 ​

文件管理系统遵循一致的 API 通信模式，具有集中的请求处理和错误管理。

函数方法参数响应处理
loadDataList()GETpageNo, pageSize, fileNameFuzzy, category, filePid更新 tableData.value
saveNameEdit()POSTfileId, filePid, fileName更新特定表格行
delFile()POSTfileIds (单个)刷新文件列表
delFileBatch()POSTfileIds (逗号分隔)刷新文件列表
moveFolderDone()POSTfileIds, filePid刷新文件列表
download()GETURL 路径中的 fileId重定向到下载 URL
